## Dynamic Content Fields

**${contact.name.first}**
- Source: Ticket purchase form, First Name field
- Fallback: "there" (as in "Hey there!")
- Example: "Jamie"

**${eventTitle}**
- Source: Wix Events Module, Event Name
- Example: "Sunrise Sessions Vol. 5"

**${eventDate}**
- Source: Wix Events Module, Event Date
- Format: Full date display

**${eventLocation}**
- Source: Wix Events Module, Event Location
- Format: Venue/address

**${calendarLinkText}**
- Auto-generated calendar add link text
- Allows users to add event to their calendar

**Ticket Download/View Link**
- Auto-generated by Wix upon ticket purchase
- Links to ticket page or PDF download
- Secure, unique URL per ticket/order

## Usage Notes

**When to Use This Email:**
- Paid ticket purchases
- After payment is confirmed/processed
- For ticketed events only
- Immediate send after purchase

**Do NOT Use For:**
- Free event registrations (use Registration Confirmation email)
- Waitlist notifications (use Waitlist email)
- Event reminders (use Reminder email)

**Brand Voice:**
- Direct and energetic
- Confident and welcoming
- Action-oriented
- Music/vibe-focused

**Tone Elements:**
- "You're in!" ‚Äì Direct, confident confirmation
- "Your tickets are ready" ‚Äì Clear, actionable
- "See you on the dancefloor" ‚Äì Music scene language
- "Come ready to vibe" ‚Äì SSS attitude
- "Grab your ticket below" ‚Äì Simple, clear CTA

## Copy Differences from Registration Email

**Registration Confirmation:**
- "You're on the list"
- No ticket download
- More explanatory about event

**Ticket Confirmation (This Email):**
- "Your tickets are ready"
- Prominent ticket download CTA
- Shorter, more action-focused
- Assumes they already know about event
